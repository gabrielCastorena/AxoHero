# AxoHero wop

**AxoHero** es un videojuego web enfocado en la estimulación cognitiva para niños, disfrazado de una divertida aventura submarina. El jugador controla a un ajolote que debe limpiar el océano resolviendo diversos minijuegos mentales.


## Áreas Cognitivas a Estimular
* Atención selectiva y reflejos.
* Memoria de trabajo (secuenciación a corto plazo).
* Control de impulsos.

## Tecnologías Usadas
* **Motor:** Unity 3D (C#)
* **Arte:** Pixel Art (LibreSprite)
* **Plataforma:** WebGL (Próximamente)
